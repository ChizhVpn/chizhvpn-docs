# Security

## Секреты
Никогда не коммитим:
- `.env`, `*.env`, `*.env.local`
- токены Telegram / ключи ЮKassa
- приватные ключи (`*.pem`, `*.key`, `id_rsa*`)
- дампы БД

Если секрет попал в репозиторий:
1) немедленно делаем ротацию (BotFather `/revoke`, ЮKassa — смена ключей)
2) вычищаем из git (лучше `git filter-repo`)
3) проверяем, что секретов больше нет

## Контакты
Если вы нашли утечку/уязвимость — лучше писать приватно владельцам проекта.
